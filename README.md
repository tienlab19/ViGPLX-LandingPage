# ViGPLX landing page

Static landing page for App Store metadata URLs.

## Files

- `index.html`: marketing landing page
- `support.html`: support URL page
- `privacy.html`: privacy policy URL page
- `assets/`: app icon and screenshots

## Suggested URLs

- Marketing URL: `https://viuniverse.vn/vigplx/`
- Support URL: `https://viuniverse.vn/vigplx/support.html`
- Privacy Policy URL: `https://viuniverse.vn/vigplx/privacy.html`

## Deploy

Upload the full `landing_page` folder to any static hosting service such as GitHub Pages, Vercel, Netlify, Cloudflare Pages, or your existing website host.
